**Critical Info for New Agent**
- **Environment Variables on Render are CRITICAL**: Nearly every major bug in this session was ultimately traced back to missing or incorrect environment variables on Render's backend or frontend services (for MongoDB, Firebase, Email, etc.). Always verify them first when debugging the live site.
- **Code Deployment Workflow**: The user must click Save to Github in the Emergent chat. This pushes the code, which then automatically triggers a new deployment on Render. Simply restarting a Render service will redeploy the *old* code.
- **MongoDB Free Tier Limits**: The MongoDB Atlas free tier cannot perform large in-memory sorts. The  index on the  collection is essential for the chat history feature to work. If other performance issues arise, check for missing indexes.
- **Chat UI is Fragile**: The chat layout uses complex flexbox and height calculations (). Small changes can re-introduce gaps or scrolling issues. Test on both mobile and desktop views after any CSS change.
